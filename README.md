# AI Agents

This repo currently keeps two separate agent implementations:

- `agent.py` for CrewAI
- `adk_app/` for Google ADK

## Setup

Run .\setup.ps1 once to create `.venv` and install dependencies.

## Run CrewAI

After setup, run `python agent.py` from the activated virtual environment.

## Run ADK

After setup, run `adk web .` from the activated virtual environment.
